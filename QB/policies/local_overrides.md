# QB Local Overrides

Use this file only for repo-local overrides that do not conflict with UXOS core policy.

Allowed:
- Repo-specific release cadence
- Repo-specific acceptance gates
- Repo-specific implementation details

Not allowed:
- Disabling versioned changes policy
- Bypassing canon/governance checks
- Changing Chief (`UXOS/00`) vs PM (`<project>/00`) role boundaries
- Replacing QB as the required Chief/PM interface
